# 🚀 Chrome 插件安装指南

## 快速安装步骤

### 1️⃣ 准备图标文件（可选）

在 `images` 文件夹中放入三个图标文件：
- `icon16.png` (16x16)
- `icon48.png` (48x48)
- `icon128.png` (128x128)

**如果暂时没有图标**，可以跳过此步骤，插件功能不受影响。

### 2️⃣ 打开 Chrome 扩展管理页面

在 Chrome 浏览器地址栏输入：
```
chrome://extensions/
```

或者通过菜单：
```
右上角三个点 → 扩展程序 → 管理扩展程序
```

### 3️⃣ 开启开发者模式

在扩展管理页面右上角，打开「**开发者模式**」开关。

### 4️⃣ 加载插件

1. 点击左上角的「**加载已解压的扩展程序**」按钮
2. 选择本项目文件夹：`E:\WorkSpace\CascadeProjects\windsurf-project`
3. 点击「选择文件夹」

### 5️⃣ 安装完成！

你应该能看到：
- ✅ 插件出现在扩展列表中
- ✅ 名称：划线取词助手
- ✅ 状态：已启用

## 🎯 开始使用

1. 打开任意网页（例如：百度、知乎、博客等）
2. 用鼠标选中一段文字
3. 会出现一个紫色的圆形浮动按钮
4. 点击按钮，选中的文字会发送到飞书 Webhook
5. 右上角会显示「✓ 已发送到飞书」的提示

## ⚙️ 查看插件详情

点击 Chrome 工具栏中的插件图标（拼图图标），找到「划线取词助手」，可以：
- 查看使用说明
- 查看 Webhook 地址
- 确认插件运行状态

## 🐛 故障排查

### 问题：插件加载失败
**解决方案**：
- 确保选择的是正确的文件夹
- 检查 `manifest.json` 文件是否存在
- 查看错误提示信息

### 问题：按钮不显示
**解决方案**：
- 确保已选中文字（文字长度 > 0）
- 刷新网页后重试
- 检查插件是否已启用

### 问题：发送失败
**解决方案**：
- 检查网络连接
- 确认 Webhook 地址是否正确
- 查看浏览器控制台的错误信息（F12）

### 问题：图标不显示
**解决方案**：
- 这是正常的，图标是可选的
- 如需显示图标，请准备对应尺寸的 PNG 文件

## 📝 修改 Webhook 地址

如果需要修改 Webhook 地址：

1. 打开 `content.js` 文件
2. 找到第 7 行：
   ```javascript
   const WEBHOOK_URL = 'https://www.feishu.cn/flow/api/trigger-webhook/f0b419896d69c20daf099813dfcf3126';
   ```
3. 修改为你的 Webhook 地址
4. 保存文件
5. 在 `chrome://extensions/` 页面点击插件的「刷新」按钮
