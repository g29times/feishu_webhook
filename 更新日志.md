# 更新日志

## v1.1 - 2025-11-19

### 🎉 新增功能

#### 1. 网页便签功能
- 在页面右侧添加常驻笔记入口按钮
- 点击按钮展开侧边栏笔记面板
- 支持为每个网页添加独立的笔记
- 笔记与URL绑定，切换页面自动切换笔记内容
- 笔记持久化存储在浏览器本地
- 支持 `Ctrl+Enter` 快速添加笔记
- 每条笔记可单独发送到飞书或删除
- 笔记数量徽章提示

#### 2. Webhook 地址配置
- 支持在插件面板中自定义 Webhook 地址
- 输入框实时验证 URL 格式
- 配置保存到浏览器本地存储
- 支持回车键快速保存
- 保存成功/失败即时反馈

### 🔧 技术改进
- 将硬编码的 Webhook 地址改为可配置
- 优化代码结构，添加 `NoteManager` 类
- 扩展 CSS 样式，新增 342 行笔记相关样式
- 扩展 JS 逻辑，新增约 315 行笔记管理代码

### 📝 文档更新
- 更新 README.md，添加新功能说明
- 创建 DESIGN_PROPOSAL.md 设计文档
- 创建测试指南.md 测试清单
